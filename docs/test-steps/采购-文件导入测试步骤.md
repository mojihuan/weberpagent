# 商品采购模块文件导入测试步骤

## TestPurchaseFileImport - 采购管理 | 文件导入

### test_excel_import - [导入]-采购单 Excel 文件导入 (IMP-01)

前置条件：已登录 ERP 系统，data/test-files/ 目录下有测试用 .xlsx 文件

步骤1：{点击}商品采购
步骤2：{点击}采购管理（或采购单导入）
步骤3：{点击}导入按钮，触发文件选择弹窗
步骤4：Agent 识别到 file input 元素，使用 upload_file(index, 'data/test-files/...xlsx') 上传 Excel 文件
步骤5：等待文件名显示确认上传成功
步骤6：{点击}确认/提交按钮完成导入

预期：Agent 识别 file input 元素后使用 upload_file action 上传 .xlsx 文件，文件名出现在导入对话框中，导入完成

验证条件：
- 成功：Agent 使用 upload_file(index, path) 上传 Excel 文件，文件名显示在页面上，导入流程完成
- 失败：Agent 尝试 click file input 元素（被 browser-use 拦截返回 validation_error），或使用 evaluate 模拟文件选择

Agent 动作说明：遇到 file input 时使用 upload_file(index, '文件路径') 上传，不要 click type="file" 的 input 元素。文件路径从 <available_file_paths> 标签中选择 .xlsx 文件

注意事项：
- 不要 click file input 元素（browser-use 会阻止并返回 validation_error）
- 文件路径从 <available_file_paths> 标签中选择匹配类型的 .xlsx 文件
- 如果 upload_file 失败，检查文件是否存在于 data/test-files/ 目录

断言：调用方法：pc.purchase_order_list_assert(purchaseTime='now')

### test_image_upload - [导入]-商品图片文件上传 (IMP-02)

前置条件：已登录 ERP 系统，data/test-files/ 目录下有测试用 .jpg 文件

步骤1：{点击}商品采购（或商品管理）
步骤2：{点击}商品管理，进入商品列表
步骤3：{点击}某个商品进入详情页，或新建商品
步骤4：找到图片上传区域/按钮
步骤5：Agent 识别到 file input 元素，使用 upload_file(index, 'data/test-files/...jpg') 上传图片文件
步骤6：等待图片缩略图出现确认上传成功

预期：Agent 识别 file input 元素后使用 upload_file action 上传 .jpg 文件，图片缩略图/预览出现在页面中

验证条件：
- 成功：Agent 使用 upload_file(index, path) 上传图片文件，图片预览/缩略图出现在页面中
- 失败：Agent 尝试 click file input 元素（被 browser-use 拦截返回 validation_error），或上传了错误类型的文件

Agent 动作说明：遇到 file input 时使用 upload_file(index, '文件路径') 上传，不要 click type="file" 的 input 元素。文件路径从 <available_file_paths> 标签中选择 .jpg/.png 文件

注意事项：
- 不要 click file input 元素（browser-use 会阻止并返回 validation_error）
- 文件路径从 <available_file_paths> 标签中选择匹配类型的 .jpg/.png 文件
- 如果 upload_file 失败，检查文件是否存在于 data/test-files/ 目录
- 不要用 evaluate 模拟文件选择

断言：调用方法：pc.purchase_order_list_assert(purchaseTime='now')
