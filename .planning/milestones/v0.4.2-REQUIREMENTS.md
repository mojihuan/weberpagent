# Requirements: v0.4.2 人工验证断言系统

**Defined:** 2026-03-22
**Core Value:** 让 QA 用自然语言写测试用例，AI 自动执行并生成报告

## v1 Requirements

### 断言验证

- [ ] **ASSERT-01**: `sell_sale_item_list_assert` 断言方法能正确执行并返回结果
- [ ] **ASSERT-02**: 断言参数（salesOrder、articlesStateStr、saleTime）正确传递到断言方法
- [ ] **ASSERT-03**: `saleTime='now'` 能正确转换为当前时间
- [ ] **ASSERT-04**: 断言结果正确存储到 context 并在报告中展示

### Bug 修复

- [ ] **BUG-01**: 发现并记录断言执行过程中的所有 bug
- [ ] **BUG-02**: 修复影响断言执行的关键 bug

### 文档完善

- [ ] **DOC-01**: 记录断言系统的使用方式（如何配置三层参数）
- [ ] **DOC-02**: 记录常见问题和注意事项

## Out of Scope

| Feature | Reason |
|---------|--------|
| 前置条件验证 | 本次专注断言，前置条件假设已能执行 |
| 测试步骤验证 | 本次专注断言，步骤执行假设已能工作 |
| 自动化测试 | 本次为人工验证，自动化测试在后续版本 |
| 新功能开发 | 仅验证和修复，不添加新功能 |
