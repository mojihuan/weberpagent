**File 1: `frontend/src/utils/reasoningParser.ts`**

Create a pure TypeScript utility with this exact interface:

```typescript
export interface ReasoningSegment {
  label: string   // 'Eval' | 'Verdict' | 'Memory' | 'Goal' | '' (empty for unlabeled)
  content: string
}

export function parseReasoning(text: string): ReasoningSegment[]
```

Implementation rules:
- If `!text || !text.trim()`, return empty array `[]`
- Split by `" | "` (pipe with surrounding spaces) into parts
- For each part, match `/^(Eval|Memory|Goal)\s*:\s*(.*)/i` (case-insensitive, per D-02)
- If label is "Eval" (case-insensitive match), additionally scan the value for embedded `Verdict:` using `/Verdict\s*:\s*(.*)/i`
  - If Verdict found: extract Eval content before the Verdict match, push `{ label: 'Eval', content: evalBeforeVerdict }` (only if non-empty after trim), then push `{ label: 'Verdict', content: verdictValue }`
  - If no Verdict: push `{ label: 'Eval', content: value }`
- For Memory/Goal: push `{ label: matchedLabel (preserve original casing from regex group 1), content: value }`
- For non-matching parts that are non-empty after trim: push `{ label: '', content: part.trim() }` (per D-07)
- Capitalize the first letter of label: "eval" -> "Eval", "memory" -> "Memory", "goal" -> "Goal" (use `label.charAt(0).toUpperCase() + label.slice(1).toLowerCase()`)
