## Excel Template Visual Contract

Phase 70's primary visual deliverable is the .xlsx template itself. The template has specific visual and structural requirements.

### Sheet 1: "测试用例" (Test Cases)

**Primary focal point:** The blue-500 (#3B82F6) header row, which immediately distinguishes column names from data rows and anchors the visual hierarchy of the entire sheet.

**Column contract (single source of truth):**

| Col | Header | Width | Type | Required | Default | Validation |
|-----|--------|-------|------|----------|---------|------------|
| A | 任务名称 | 25 | str | YES | - | min_length=1 |
| B | 任务描述 | 40 | str | YES | - | min_length=1 |
| C | 目标URL | 35 | str | NO | "" | - |
| D | 最大步数 | 12 | int | NO | 10 | 1-100 dropdown |
| E | 前置条件 | 40 | JSON | NO | None | JSON array parseable |
| F | 断言 | 50 | JSON | NO | None | JSON array parseable |

**Header row styling:**
- Font: bold, white text on blue-500 background (#3B82F6)
- Alignment: center horizontal, center vertical
- Border: thin border all sides

**Example data (rows 2-3):**
- Row 2: Full data example (all columns filled including JSON)
- Row 3: Minimal data example (only required fields, optional columns empty)

**Data validation:**
- Column D (最大步数): whole number between 1-100, allow blank
- Error message: "请输入 1-100 之间的整数"
- Input prompt: "请输入 1 到 100 之间的整数"
- Validation range: D2:D10000

**Functional features:**
- Freeze panes at A2 (header row stays visible while scrolling)
- No merged cells in the template

### Sheet 2: "README" (Instructions)

**Content sections:**
1. Overview: What this template is for
2. Column descriptions: Each column name, type, required/optional, format rules
3. JSON format examples: How to write preconditions and assertions
4. Tips: Common mistakes, how to handle empty values
5. Contact: How to get help

**Language:** Chinese only (target audience is Chinese-speaking QA testers).

**Source:** CONTEXT.md D-01 through D-05 (column format decisions), RESEARCH.md Column Contract table, ROADMAP.md Phase 70 success criteria.
