  <implementation>
PreSubmitGuard is a @dataclass with:
- `_task: str` (stored for expectation extraction, set via setter or constructor)
- `_expectations: dict[str, str]` (cached extracted expectations)

GuardResult is a @dataclass(frozen=True):
- `should_block: bool`
- `message: str`

EXPECTATION_PATTERNS: list of compiled regex patterns for extracting expected values from task:
```python
EXPECTATION_PATTERNS = [
    (re.compile(r'销售金额[：:]*\s*(\d+(?:\.\d+)?)\s*元?'), '销售金额'),
    (re.compile(r'物流费用[：:]*\s*(\d+(?:\.\d+)?)\s*元?'), '物流费用'),
    (re.compile(r'金额[：:]*\s*(\d+(?:\.\d+)?)\s*元?'), '金额'),
    (re.compile(r'付款状态[：:]*\s*(已付款|未付款|部分付款|待付款)'), '付款状态'),
]
```

SUBMIT_KEYWORDS: set of Chinese/English submit button text:
```python
SUBMIT_KEYWORDS = {'确认', '提交', '保存', '确定', 'submit', 'confirm', 'save', 'ok'}
```

`check(self, action_name: str, target_index: int | None, task: str, actual_values: dict[str, str] | None, submit_button_text: str | None) -> GuardResult`:
1. If action_name != "click", return GuardResult(should_block=False, message="")
2. Extract expectations from task using EXPECTATION_PATTERNS
3. If no expectations found, return GuardResult(should_block=False, message="") (per MON-06)
4. If actual_values is None, return GuardResult(should_block=False, message="") (cannot verify without browser)
5. Check if submit_button_text contains any SUBMIT_KEYWORDS
6. If not a submit button, return GuardResult(should_block=False, message="")
7. Compare each expectation against actual_values
8. If any mismatch, return GuardResult(should_block=True, message=validation_report)
9. If all match, return GuardResult(should_block=False, message="")

Message format (per D-05, D-06): Chinese with structured format:
- Blocked: `【提交拦截】检测到提交操作，但以下字段值不匹配：{field_details}。请先修正字段值再提交。`
- Pass: empty string

`_extract_expectations(self, task: str) -> dict[str, str]`:
- Iterate EXPECTATION_PATTERNS, find first match in task
- Return dict mapping field name to extracted value
  </implementation>
