## 1. 需求概述
- 项目: aiDriveUITest (AI 驱动的 UI 自动化测试平台)
- 技术要求: 需运行 Playwright Chromium 浏览器
- 预算约束: 100元/月以下
- 操作系统: Ubuntu 22.04 (Playwright 兼容性最佳)

## 2. 云厂商对比

### 2.1 阿里云轻量应用服务器
| 配置 | CPU | 内存 | 带宽 | 存储 | 月费 |
|------|-----|------|------|------|------|
| 入门型 | 2核 | 2GB | 3Mbps | 60GB SSD | ~50元 |
| **推荐** | **2核** | **4GB** | **4Mbps** | **60GB SSD** | **~16.6-25元(新人)** |

新人优惠链接: https://www.aliyun.com/activity/new
- 优势: 新人优惠力度大，性价比最高，国内市场占有率第一
- 劣势: 需要 ICP 备案 (国内地域)

### 2.2 腾讯云轻量应用服务器
| 配置 | CPU | 内存 | 带宽 | 存储 | 月费 |
|------|-----|------|------|------|------|
| 入门型 | 2核 | 2GB | 4Mbps | 50GB SSD | ~40元 |
| 标准型 | 2核 | 4GB | 6Mbps | 60GB SSD | ~70元 |

新人优惠链接: https://cloud.tencent.com/act/new
- 优势: 带宽配置较高
- 劣势: 价格略高于阿里云

### 2.3 华为云 HECS 云耀服务器
| 配置 | CPU | 内存 | 带宽 | 存储 | 月费 |
|------|-----|------|------|------|------|
| 入门型 | 2核 | 2GB | 3Mbps | 40GB SSD | ~45元 |
| 标准型 | 2核 | 4GB | 4Mbps | 60GB SSD | ~80元 |

- 优势: 企业级服务
- 劣势: 价格最高，新人优惠较少

## 3. 推荐方案

**推荐: 阿里云轻量应用服务器 2核4G**

理由:
1. 价格最低 (新人优惠约 16.6-25元/月)
2. 4GB 内存满足 Playwright 运行需求
3. 60GB SSD 存储足够存放测试数据和截图
4. 国内市场占有率第一，文档丰富

推荐配置:
- 地域: 华东/华南 (国内，需备案) 或 香港/新加坡 (免备案，延迟稍高)
- 规格: 2核 CPU / 4GB 内存 / 4Mbps 带宽
- 存储: 60GB SSD
- 系统: Ubuntu 22.04 LTS
- 付费: 按月付费

## 4. 购买步骤

1. 访问阿里云新人活动页面: https://www.aliyun.com/activity/new
2. 登录/注册阿里云账号
3. 选择「轻量应用服务器」
4. 选择配置: 2核4G / 60GB SSD / 4Mbps
5. 选择地域: 华东1(杭州) 或 华南1(深圳)
6. 选择镜像: Ubuntu 22.04 LTS
7. 选择付费周期: 1个月 (按月付费)
8. 确认订单并支付

## 5. 安全组配置

购买后需配置安全组开放端口:
- SSH: 22 (管理登录)
- HTTP: 80 (Nginx)
- HTTPS: 443 (Nginx SSL)
- API: 8080 (FastAPI 后端，可选，建议仅内网)

## 6. 后续步骤

购买完成后:
1. 记录服务器公网 IP 地址
2. 使用 SSH 登录验证: `ssh root@<公网IP>`
3. 进入 Phase 38 部署执行
