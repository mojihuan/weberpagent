# 云服务器性价比调研报告

**项目:** aiDriveUITest (AI 驱动的 UI 自动化测试平台)
**生成日期:** 2026-03-23
**状态:** 待购买

---

## 1. 需求概述

### 1.1 项目背景

aiDriveUITest 是一个 AI 驱动的 UI 自动化测试平台，让 QA 测试人员使用自然语言编写测试用例。基于 Browser-Use + 阿里云 Qwen 3.5 Plus 构建，实现「自然语言描述 → AI 决策 → Playwright 执行」的自动化测试流程。

### 1.2 技术要求

| 项目 | 要求 |
|------|------|
| 浏览器 | Playwright Chromium (需要 GUI 环境或 headless 模式) |
| 内存 | 最低 4GB (Playwright + Chromium + Python 应用) |
| CPU | 最低 2 核 |
| 存储 | 60GB+ (测试数据、截图、日志) |
| 操作系统 | Ubuntu 22.04 LTS (Playwright 兼容性最佳) |

### 1.3 预算约束

**预算上限:** 100 元/月

### 1.4 部署架构

```
┌─────────────────────────────────────────────────────────┐
│                    云服务器 (Ubuntu 22.04)                │
├─────────────────────────────────────────────────────────┤
│  Nginx (反向代理 + 静态资源)                              │
│    ├── 80/443 → 前端 React 应用                          │
│    └── /api → 后端 FastAPI                               │
├─────────────────────────────────────────────────────────┤
│  FastAPI + Gunicorn                                      │
│    ├── 8080 → API 服务                                   │
│    └── SSE → 实时进度推送                                │
├─────────────────────────────────────────────────────────┤
│  Playwright Chromium (Headless)                          │
│    └── AI 自动化测试执行                                  │
├─────────────────────────────────────────────────────────┤
│  SQLite (WAL 模式)                                       │
│    └── 持久化存储                                        │
└─────────────────────────────────────────────────────────┘
```

## 4. 推荐方案

### 4.1 最终推荐

**推荐: 阿里云轻量应用服务器 2核4G**

### 4.2 推荐理由

| 序号 | 理由 | 说明 |
|------|------|------|
| 1 | **价格最低** | 新人优惠约 16.6-25 元/月，远低于预算上限 |
| 2 | **内存充足** | 4GB 内存满足 Playwright + Chromium + Python 应用运行需求 |

| 4 | **生态完善** | 国内市场占有率第一，文档丰富，问题易解决 |

## 5. 购买步骤

### 5.1 购买前准备

1. 准备一个未注册过阿里云的手机号 (用于新人优惠)
2. 准备身份证 (实名认证)
3. 准备支付宝或银行卡 (支付)

### 5.2 详细购买步骤

#### 步骤 1: 访问新人活动页面

访问: https://www.aliyun.com/activity/new

#### 步骤 2: 登录/注册阿里云账号

- 如果是新用户，使用手机号注册
- 完成实名认证 (需要身份证)

#### 步骤 3: 选择轻量应用服务器

在活动页面找到「轻量应用服务器」卡片，点击立即购买

#### 步骤 4: 选择配置

| 选项 | 选择值 |
|------|--------|
| 地域 | 华东1(杭州) 或 华南1(深圳) |
| 规格 | 2核4GB |
| 存储 | 60GB SSD (默认) |
| 带宽 | 4Mbps (默认) |

#### 步骤 5: 选择镜像

选择「系统镜像」→「Ubuntu」→「Ubuntu 22.04 LTS」

#### 步骤 6: 选择付费周期

选择「1个月」或「3个月」(新人优惠通常是 3 个月起)

#### 步骤 7: 确认订单

检查配置和价格，确认无误后点击「立即购买」

#### 步骤 8: 支付

选择支付方式 (支付宝/银行卡)，完成支付

### 5.3 购买后确认

支付成功后:

1. 记录服务器公网 IP 地址
2. 记录登录密码 (或通过控制台重置)
3. 确认服务器状态为「运行中」

## 6. 安全组配置

购买完成后需要配置安全组，开放必要的端口:

### 6.1 必需端口

| 端口 | 协议 | 用途 | 访问范围 |
|------|------|------|----------|
| 22 | TCP | SSH 管理登录 | 建议限制 IP |
| 80 | TCP | HTTP (Nginx) | 0.0.0.0/0 |
| 443 | TCP | HTTPS (Nginx SSL) | 0.0.0.0/0 |

### 6.2 可选端口

| 端口 | 协议 | 用途 | 访问范围 |
|------|------|------|----------|
| 8080 | TCP | FastAPI 后端 (开发调试) | 建议仅内网或限制 IP |

### 6.3 配置步骤

1. 登录阿里云控制台
2. 进入「轻量应用服务器」→ 选择服务器
3. 点击「安全」→「防火墙」
4. 点击「添加规则」，添加上述端口

## 7. 首次登录验证

### 7.1 SSH 登录

```bash
ssh root@<公网IP>
```

首次登录会提示确认指纹，输入 `yes` 继续。

### 7.2 基础环境检查

```bash
# 检查系统版本 (应为 Ubuntu 22.04)
lsb_release -a

# 检查内存 (应显示 4GB)
free -h

# 检查磁盘 (应显示 60GB)
df -h

# 检查 CPU (应显示 2 核)
nproc
```

### 7.3 基础环境准备

```bash
# 更新系统
apt update && apt upgrade -y

# 安装常用工具
apt install -y curl wget git vim

# 安装 Python 3.11 (如果未安装)
apt install -y python3.11 python3.11-venv python3-pip
```
