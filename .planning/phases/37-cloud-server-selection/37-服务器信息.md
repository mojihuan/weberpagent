## 配置步骤 (阿里云控制台)

1. 登录阿里云控制台: https://ecs.console.aliyun.com/
2. 进入「轻量应用服务器」
3. 点击服务器实例
4. 选择「安全」→「防火墙」
5. 添加规则开放上述端口
