**PreconditionService.substitute_variables** (backend/core/precondition_service.py:276-301):
```python
@staticmethod
def substitute_variables(text: str, context: dict[str, Any]) -> str:
    if not text or '{{' not in text:
        return text

    env = Environment(
        variable_start_string='{{',
        variable_end_string='}}',
        undefined=StrictUndefined,
    )
    template = env.from_string(text)
    return template.render(**context)
```

Uses Jinja2 with StrictUndefined - undefined variables raise UndefinedError.
