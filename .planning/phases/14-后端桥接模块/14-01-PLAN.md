Create `backend/core/external_precondition_bridge.py` with the following structure:

1. **Module-level singleton state** (top of file):
   ```python
   _pre_front_class = None
   _import_error = None
   _operations_cache: dict[str, str] | None = None
   _modules_cache: list[dict] | None = None
   _path_configured = False
   ```

2. **configure_external_path(weberp_path: str | None) -> tuple[bool, str]**:
   - If weberp_path is None, return (True, "WEBSERP_PATH not configured (optional feature)")
   - Use Path(weberp_path).resolve() to get absolute path
   - Check path exists and is directory
   - Add to sys.path only if not already present (idempotent)
   - Set _path_configured = True on success

3. **load_pre_front_class() -> tuple[type | None, str | None]**:
   - Return cached _pre_front_class if not None
   - Return (None, _import_error) if previous import failed
   - Try: `from common.base_prerequisites import PreFront`
   - On ImportError: set _import_error to message including "Ensure config/settings.py exists in webseleniumerp"
   - On success: set _pre_front_class = PreFront, return (PreFront, None)

4. **is_available() -> bool**:
   - Call load_pre_front_class()
   - Return True if class is not None

5. **get_unavailable_reason() -> str | None**:
   - Return None if available
   - Return error message or "External module not configured"

6. **_parse_operations_from_source() -> tuple[dict[str, str], list[dict]]**:
   - Return cached (_operations_cache, _modules_cache) if both not None
   - Load PreFront class, return ({}, []) if unavailable
   - Use `inspect.getsource(PreFront.operations)` to get source code
   - Parse line by line:
     - Module comment format: `# 配件管理|配件采购|新增采购` -> extract first two parts as "配件管理 - 配件采购"
     - Operation code format: `'FA1': [self.request...]` -> extract code with regex `r"'([A-Z0-9@]+)'"`
     - Extract inline description from `# description` after operation code
   - Build modules_dict: {module_name: [{code, description}, ...]}
   - Cache and return (operations, modules_list)

7. **get_available_operations() -> dict[str, str]**:
   - Return operations dict from _parse_operations_from_source()

8. **get_operations_grouped() -> list[dict]**:
   - Return modules list from _parse_operations_from_source()
   - Format: [{"name": "配件管理 - 配件采购", "operations": [{"code": "FA1", "description": "..."}]}]

9. **generate_precondition_code(operation_codes: list[str], weberp_path: str) -> str**:
   - Generate Python code string with:
     ```python
     import sys
     sys.path.insert(0, '<WEBSERP_PATH>')

     from common.base_prerequisites import PreFront

     pre_front = PreFront()
     pre_front.operations(['FA1', 'HC1'])

     context['precondition_result'] = 'success'
     ```

10. **reset_cache()** (for testing):
    - Reset all module-level globals to initial state

Import requirements:
- `import sys, logging, inspect, re`
- `from pathlib import Path`
- `from typing import Any`
- `logger = logging.getLogger(__name__)`

CRITICAL: Do NOT import from backend.config at module level (causes circular imports). Import get_settings inside functions that need it.
