    1. 显示当前 remote: `git remote -v`

    5. 实际推送到新仓库: `git push -u origin main`

    ```bash
    # 替换 remote URL
    git remote set-url origin <NEW_REPO_URL>

    # 验证
    git remote -v

    # 推送
    git push -u origin main
    ```
  </action>
